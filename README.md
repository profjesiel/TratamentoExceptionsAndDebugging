# TratamentoExceptionsAndDebugging

Repositório criado para registrar estudos, anotações e certificado do módulo **Debugging e Tratamento de Exceções em Java**, realizado no **Bootcamp NTT DATA: Backend Java com Spring AI**.

## 📚 Sobre o projeto

Este repositório reúne conteúdos relacionados ao aprofundamento em **Debugging** e **Tratamento de Exceções em Java**, abordando conceitos fundamentais para o desenvolvimento de aplicações mais seguras, organizadas e robustas.

Durante os estudos, foram exploradas práticas importantes para identificar erros, analisar o fluxo de execução do código e implementar mecanismos adequados de tratamento de falhas em aplicações Java.

## 🚀 Conteúdos abordados

* Conceitos de **Debugging** em Java
* Identificação e análise de erros durante a execução
* Uso de breakpoints e inspeção de variáveis
* Tratamento de exceções com `try`, `catch` e `finally`
* Diferença entre `throw` e `throws`
* Criação e propagação de exceções
* Boas práticas no tratamento de erros
* Escrita de código mais seguro, legível e manutenível

## 🎯 Objetivo

O objetivo deste repositório é documentar o aprendizado obtido no bootcamp e servir como material de apoio para revisões futuras sobre:

* depuração de código Java;
* tratamento adequado de exceções;
* prevenção de falhas e melhoria da qualidade do software;
* fortalecimento da base para desenvolvimento backend.
